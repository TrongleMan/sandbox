Best Practices

Every Claude Product

Tips and workflows to get the most out of each product, whether you're using Claude personally or at work.

🌐

Claude.ai

The main web interface. Where most people start — and where a lot of power users live.

Best Practices
  • Use Projects to maintain context across long conversations — no re-explaining every session
  • Set a custom system prompt in Project settings to define tone, format, and persona
  • Upload files, images, and PDFs — Claude can analyze, summarize, and extract information directly
  • Browse the Prompt Library when you're not sure how to ask for what you need
  • Use the Artifacts panel to generate and refine code, documents, and diagrams in-session
Open Claude.ai →
💻

Claude for Desktop

Native Mac and Windows app. Faster, integrates with your OS, and supports computer use.

Best Practices
  • Enable computer use (research preview, Pro/Max) to let Claude operate your desktop autonomously
  • Grant local file system access so Claude can read and write files without copy-paste
  • Use the global keyboard shortcut to summon Claude from anywhere without switching windows
  • Pair with Projects for persistent context that carries over every time you open the app
  • Great for long research sessions — the desktop app handles large file uploads more smoothly
Download the App →
📱

Claude Mobile

iOS and Android. Full Claude in your pocket with voice, camera, and share-sheet integration.

Best Practices
  • Use Voice mode for commute conversations, brainstorming walks, or hands-free journaling
  • Point your camera at anything — menus, whiteboards, labels, books — and ask Claude questions about it
  • Use the iOS/Android share sheet to send articles, screenshots, or text straight to Claude from any app
  • Your Projects sync across mobile and desktop — start on your phone, continue on your computer
  • Voice conversations feel more like talking through a problem than typing — great for early-stage thinking
Get the App →
⚡

Claude Code

Agentic coding in your terminal. Runs tasks, writes code, edits files, and commits — autonomously.

Best Practices
  • Add a CLAUDE.md file to your project root with codebase context, conventions, and patterns
  • Describe the goal, not just the step — Claude Code performs better with outcomes than instructions
  • Use parallel sub-agents to run independent tasks simultaneously within the same project
  • Set up hooks to automate pre/post actions on every file edit or test run
  • Lean on git to checkpoint progress — Claude Code integrates with version control natively
Learn About Claude Code →
🏢

Claude for Work

Team and Enterprise plans. Shared Projects, admin controls, compliance features, and analytics.

Best Practices
  • Set up shared Projects so your whole team benefits from the same persistent context
  • Create department-specific Projects with tailored system prompts per team (Sales, Engineering, Support)
  • Use the admin console to see usage patterns and identify high-value workflows to standardize
  • Enterprise plans include SSO, data residency controls, and compliance features for regulated industries
  • Claude Code is now available on Team and Enterprise plans with usage analytics

Everyday Workflows

What Could You Do With This?

Not sure where to start? Here are twelve ways real people use Claude in their daily lives — no technical background required.

The best way to learn Claude is to throw something real at it. Pick a task from your actual life or work — something you've been putting off, something you're stuck on, or just something tedious. The prompts below are starting points, not scripts. Claude responds best when you give it context about who you are and what you actually need.

✍️

Writing & Editing

First drafts, rewrites, feedback, and formatting. Claude can help you write clearer, faster — and explain why the changes work.

Try asking "Here's a draft of my performance review. Make it more specific and results-focused, and cut it by 20%."
🔍

Research Shortcuts

Paste in long documents, reports, or links and ask Claude to extract, summarize, or compare. Saves hours on dense reading.

Try asking "Here's a 40-page contract. Pull out every clause related to termination, liability, and payment terms."
📚

Learning Anything

Claude is a patient, adaptive tutor. Ask it to explain something at your level, then push until you actually understand it.

Try asking "Explain how compound interest works using a real example with actual numbers. I'm 28 and just opened a Roth IRA."
✈️

Trip Planning

Claude can build full itineraries with timing, logistics, and alternatives. Give it constraints and it handles the tradeoffs.

Try asking "Plan 7 days in Japan for two adults who love food, architecture, and don't want tourist traps. Budget: moderate."
💡

Brainstorming

Use Claude as a thinking partner. It won't just generate a list — it'll push back, challenge assumptions, and help you refine ideas.

Try asking "I want to launch a side project. Here are three ideas. Help me stress-test each one and tell me which has the best angle."
📧

Email Wrangling

From difficult replies to inbox zero strategy. Paste the thread, explain the dynamic, and let Claude draft the response.

Try asking "Here's an email from a client who's clearly frustrated but being polite about it. Help me respond in a way that addresses the real concern."
🍳

Meal Planning

Weekly dinners, grocery lists, meal prep schedules — all with your dietary constraints, budget, and actual cooking skill level factored in.

Try asking "Plan 5 weeknight dinners for two people. Budget: under $80. One person doesn't eat red meat. I hate doing dishes."
💪

Health & Fitness

Workout plans, nutrition questions, recovery guidance. Be specific about your starting point and Claude will meet you there.

Try asking "I want to start running but I get shin splints. Design a 6-week beginner plan that builds up slowly and prevents injury."
💰

Financial Clarity

Untangle financial concepts, stress-test decisions, build simple models. Claude explains the tradeoffs without the jargon.

Try asking "I have $8,000 in savings. Walk me through whether to pay off my car loan early or invest it, given these numbers: [paste details]."
🏠

Home Projects

DIY guidance, contractor questions, renovation planning. Claude helps you understand what you're getting into before you start.

Try asking "I want to tile my bathroom floor myself. I've never done it before. What do I actually need to know before I start?"
👔

Career Help

Resumes, cover letters, interview prep, LinkedIn rewrites. Tell Claude the role and they'll tailor everything to the context.

Try asking "Here's my resume. Here's the job description. Rewrite my experience section to match what they're actually asking for."
🎨

Creative Projects

Writing, worldbuilding, scripts, song lyrics, design concepts. Claude collaborates — it'll riff, refine, and follow your vision.

Try asking "I want to write a short horror story set in a grocery store at 3am. Help me develop the atmosphere and first scene."
